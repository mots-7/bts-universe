About the website

Wether you just discovered BTS or you've been a fan for a while, keeping up with all the content can be overwhelming sometimes.

BTS UNIVERSE is an extremely basic guide for people who are getting to know BTS: you'll find all the basic info about the members, discography, latest news about the group and more!

If you're not new to the fandom, you can use this website to:

- find a list of some BTS related content, official or fanmade, that I recommend (here)
